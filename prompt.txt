    },
    {

    },
    {

      "content": "текст блока с информацией о марке стали, кодах, стандартах",
      "start_line": 21,
      "end_line": 30
    },
    {
      "block_type": "chemical_composition",

=== ПРОМПТ DOCUMENT_INFO ===
Извлеки базовую информацию о документе из предоставленного текста.

Верни результат в формате JSON:
{
  "document_info": {
    "document_number": "номер документа",
    "date": "дата документа", 
    "addressees": ["список адресатов"]
  }
}

=== ПРОМПТ DOCUMENT_TITLE ===
Извлеки заголовок и метаданные документа из предоставленного текста.

Верни результат в формате JSON:
{
  "document_title": {
    "document_name": "Наименование документа",
    "document_subname": "Поднаименование документа",
    "order_requirement": "Требования к выполнению заказа",
    "variant_naumber": "Номер варианта",
    "code_OEMK": "Код ОЭМК",
    "code_production": "Код продукции",
    "export_name": "Экспортное наименование",
    "conventional_name": "Условный аналог",
    "smelting_standart": "Стандарт выплавки",
    "consumer_property": "Номер потребительских свойств"
  }
}

=== ПРОМПТ STEEL_INFO ===
Извлеки информацию о стали из предоставленного текста.

Верни результат в формате JSON:
{
  "steel_info": {
    "steel_grade": "марка стали",
    "codes": {"тип_кода": "значение"},
    "standards": ["список стандартов"],
    "export_name": "экспортное наименование",
    "analog": "условный аналог"
  }
}

=== ПРОМПТ CHEMICAL_COMPOSITION ===
Извлеки химический состав из предоставленного текста.

Верни результат в формате JSON:
{
  "chemical_composition": {
    "recommended": {"элемент": "значение"},
    "certificate": {"элемент": "значение"}, 
    "espz": {"элемент": "значение"}
  }
}

=== ПРОМПТ TECHNICAL_REQUIREMENTS ===
Извлеки технические требования из предоставленного текста.

Верни результат в формате JSON:
{
  "technical_requirements": {
    "production_process": "схема производства",
    "notes": ["список примечаний"],
    "requirements": [{"number": "номер", "text": "текст требования"}]
  }
}

=== ПРОМПТ TABLES ===
Извлеки таблицы из предоставленного текста.

Верни результат в формате JSON:
{
  "tables": [
    {
      "table_number": "номер таблицы",
      "type": "тип таблицы",
      "data": "данные таблицы"
    }
  ]
}

=== ОБЩИЕ ПРАВИЛА ===
1. Если информация не найдена, используй null или пустые массивы
2. Сохраняй точные значения из документа
3. Если встречаются OCR ошибки, попробуй их исправить логически
4. Возвращай только валидный JSON без дополнительного текста
5. Для ПРОМПТ MAIN: разделяй текст на логические блоки по смыслу
6. Для специализированных промптов: извлекай только соответствующие данные
