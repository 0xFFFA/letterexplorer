# Простые инструкции для LLM

Проанализируй технический документ и найди следующие данные:

## 1. ОСНОВНАЯ ИНФОРМАЦИЯ

- **Номер документа** (переменная: document_number) - ищи формат ЦЦ-ЦЦЦЦ
- **Дата** (переменная: date) - ищи формат ММ.ГГГГ
- **Адресаты** (переменная: addressees) - строки "Начальнику ..."

## 2. ДАННЫЕ О СТАЛИ

- **Марка стали** (переменная: steel_grade) - после "стали марки"
- **Экспортное название** (переменная: export_name) - после "экспортное наименование"
- **Аналог** (переменная: analog) - после "условный аналог"

## 3. ХИМИЧЕСКИЙ СОСТАВ

Найди таблицу с химическими элементами (С, Si, Mn, S, Cr, Ni, Cu, Al).
Извлеки значения для:
- Рекомендуемых значений
- Аттестатных значений
- ЭСПЦ значений

## ЗАДАЧА

Для КАЖДОГО найденного элемента:
1. Найди значение в тексте
2. Создай regex паттерн для извлечения этого значения
3. Верни JSON с данными и regex

## ФОРМАТ ОТВЕТА

```json
{
  "sections": {
    "раздел": {
      "extracted_data": { "переменная": "значение" },
      "regex_patterns": { 
        "переменная": {
          "pattern": "твой regex",
          "flags": ["MULTILINE"],
          "description": "описание"
        }
      }
    }
  }
}
```

Верни ТОЛЬКО JSON без markdown блоков!
