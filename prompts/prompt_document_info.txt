Извлеки базовую информацию о документе из предоставленного текста.

Верни результат в формате JSON:
{
  "document_info": {
    "document_number": "номер документа",
    "date": "дата документа", 
    "addressees": ["список адресатов"]
  }
}

ВАЖНО:
1. Если информация не найдена, используй null или пустые массивы
2. Сохраняй точные значения из документа
3. Если встречаются OCR ошибки, попробуй их исправить логически
4. Возвращай только валидный JSON без дополнительного текста
